# Contributing

日本語でのコミュニケーション歓迎です。

## Issue

バグ報告や機能要望は Issue で受け付けています。お気軽にどうぞ。

## Pull Request

PRを送る前に、まず Issue で相談してください。方向性を合わせてから作業に入るほうがお互い効率的です。

## エージェント定義のカスタマイズ共有

エージェント定義（`agents/*.md`）のカスタマイズ例を共有したい場合は、Discussion での投稿を推奨します。他のユーザーの参考になります。
